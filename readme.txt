This project contains two files:
 1. people_in_a_circle_problem.py : This is the main solution file.
 2. Test_people_in_a_circle_problem.py : This is the unittest file for the solution.


 # How to run the program solution:
 1. Open command prompt
 2. Navigate to the parent directory of the project
 3. execute command "python people_in_a_circle_problem n", where n is the number of people ina circle


 # How to run the unittest:
 1. Open command prompt
 2. Navigate to the parent directory of the project
 3. execute command "python -m unittest Test_people_in_a_circle_problem.py".
